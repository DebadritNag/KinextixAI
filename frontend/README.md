# Kinetix AI — Flutter Web Frontend

This is the Flutter web frontend for the Kinetix AI supply chain optimization system.

For full documentation including setup instructions, architecture, API reference, and design system details, see the **[project README](../README.md)** in the root directory.

## Quick Start

```bash
# Install dependencies
flutter pub get

# Run in Chrome (backend must be running on localhost:8000)
flutter run -d chrome

# Run tests
flutter test
```

> The backend must be running before the frontend will work. See the root README for backend setup instructions.
